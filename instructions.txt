--------- Instructions to create external catalog -------------

Step 01 : Create an S3 bucket in your AWS account.

Step 02 : Create an external location in your databricks account.

Step 03 : Add the location of the S3 bucket created in step 2 --> Generate token --> Copy the token  -> Launch QuickStart

Step 04 : Under parameters section, paste the databricks personal access account token --> Click on Create Stack 

Step 05 : Create a new catalog --> Assign a name, Set the type to Standard and choose the external location (S3 bucket created in Step - 1). Click Create 

----------- Create a normal catalog -----------------

Step 06 : Create new catalog --> Type - Standard --> Click on Create 

Step 07 : Inside the catalog created in step 6, inside default schema, create a managed volume

Step 08 : Create 2 directories inside the managed volume (named as data) - customer_data and bookings_data 

Step 09 : Upload the bookings and customer data into the directories
